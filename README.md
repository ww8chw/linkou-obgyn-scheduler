# 林口長庚婦產科住院醫師排班計算器

每月計算各班別、各職級、每人應值幾平幾假（只算數量分配，不排逐日班表）。

## 線上使用

直接開啟網站即可（純前端、可離線）：

👉 **https://ww8chw.github.io/linkou-obgyn-scheduler/**

或下載 `index.html` 用瀏覽器打開，也可用 LINE／email 傳檔分享。

## 使用方式

1. **選月份** — 自動帶入 2026 台灣官方假日計算的平日／假日天數（可手動修改）。
2. **填台北輪訓人數**（值 T1／T2）。
3. **填林口各職級**人數、大／小特休、基隆籍；R3 另填可值總值班人數與班數上限。
4. **填外訓扣除後剩餘需求** — PGY1 月底最後一天不能值的病房班洞數，以及扣除童綜合外訓後的 L5（平／假各填）。
5. 按 **計算**，下方顯示各職級結果與警示。

警示區會在人力不足／超載／假日班分配失衡時以紅色列出，需人工微調。

## 排班規則重點

- 班別計算順序：病房班 → L5 → L1 → L → T1 → T2 → LR，各依職級順位填補。
- 硬性上限：每人 ≤ 8 班、≤ 3 假（5平3假／6平2假皆可），絕不超過。
- 特休：大特休 −2 平日、小特休 −1 平日（假日不減）；缺額同級吸收，超過 8 班自動往上一職級。
- 基隆籍不值 T1／T2；台北輪訓只值 T1／T2。
- R3 的 L 班受「可值人數」與「班數上限」限制。

## 開發

純 JavaScript，無外部相依，使用 Node 內建測試。

```bash
# 跑測試
node --test tests/holidays2026.test.js tests/scheduler.test.js tests/smoke.test.js

# 由 src/ 重新打包出單一檔 index.html
node build.js
```

檔案結構：

- `src/holidays2026.js` — 2026 台灣官方假日資料與當月平假天數計算
- `src/scheduler.js` — 核心排班演算法
- `src/ui.js`、`src/index.template.html` — 介面（開發用）
- `build.js` — 將上述內聯成單一 `index.html`
- `index.html` — 最終產物（部署用）

> 假日資料目前只內建 2026 年。其他年份需依行政院人事行政總處公告補上。
